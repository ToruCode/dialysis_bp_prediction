# Dialysis_Bp_Prediction
透析中の血圧を予測するAIモデルの実装とアプリケーション

## 概要
透析中の収縮期血圧・拡張期血圧・脈拍を予測する多出力回帰モデル。

## データ
※プライバシー保護のため数値は改変済み。

## パイプライン
1. 前処理（StandardScaler、train/val/test分割）
2. 多層パーセプトロンによる学習
3. RMSE/MAE/R2評価
4. Permutation Importanceによる解釈

## モデル構造
- Linear(64) → BN → Dropout
- Linear(32) → BN → Dropout
- Linear(3)

## 結果

テストデータにおける評価結果は以下の通り。

| 指標 | 収縮期 | 拡張期 | 脈拍 |
|------|--------|--------|------|
| RMSE | 27.85 mmHg | 14.17 mmHg | 3.35 bpm |
| MAE  | 22.77 mmHg | 11.55 mmHg | 2.72 bpm |
| R2   | -0.086 | -0.080 | -0.012 |

本モデルでは、現時点では十分な予測性能は得られていない。
特にR2が負値であることから、ベースライン（平均予測）を上回る精度には至っていない。

今後の改善点として、
- 特徴量設計の見直し
- モデル構造の再検討
- ハイパーパラメータ最適化
- データ量の増加

を検討している。

※本データはプライバシー保護のため数値を改変しているため、
実臨床データと同等の精度評価は行っていない。
